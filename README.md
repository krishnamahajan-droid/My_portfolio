## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/krishnamahajan-droid/My_portfolio.git
```

Navigate to the project folder

```bash
cd My_portfolio/My-Portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
